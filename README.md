# 📝 To-Do List



A simple and responsive **To-Do List** application built using **React.js** that helps users organize and manage their daily tasks efficiently. Users can add, edit, delete, and mark tasks as completed through an intuitive interface.



---



# 🚀 Features



* ➕ Add new tasks

* ✏️ Edit existing tasks

* 🗑️ Delete tasks

* ✅ Mark tasks as completed

* 📋 Display all tasks in an organized list

* ⚡ Real-time updates

* 📱 Responsive user interface



---



# 🛠️ Tech Stack



* **Frontend:** React.js

* **Language:** JavaScript (ES6+)

* **Markup:** HTML5

* **Styling:** CSS3

# ⚙️ Installation



## 1️⃣ Clone the Repository



```bash

git clone https://github.com/krishna9-31/todolist.git

```



## 2️⃣ Navigate to the Project Folder



```bash

cd todolist

```



## 3️⃣ Install Dependencies



```bash

npm install

```



## 4️⃣ Start the Development Server



```bash

npm start

```



The application will be available at:



```text

http://localhost:3000

```

# 📌 Usage



1. Enter a task in the input field.

2. Click the **Add** button.

3. Edit tasks whenever needed.

4. Mark completed tasks.

5. Delete tasks that are no longer required.
